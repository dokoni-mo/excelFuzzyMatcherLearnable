Excel Fuzzy Matcher with Learning 

Описание
Excel Fuzzy Matcher with Learning Capability - это инструмент для автоматического сопоставления данных, использующий алгоритмы нечеткого поиска и возможности обучения. Инструмент разработан для упрощения процесса сопоставления поисковых запросов с существующими записями базы данных и может обучаться на основе пользовательских корректировок.

Ключевые особенности
Автоматический анализ данных: Excel автоматически читает последний измененный документ в указанных папках.
Нечеткое сопоставление: Инструмент использует алгоритмы нечеткого поиска для сопоставления строк в базе данных.
Таблица сопоставления с возможностью обучения: Внутренний инструмент Power Query использует таблицу сопоставления "From-To" для повышения точности поиска. Ручной поиск упрощен с помощью отдельной формы VBA.
Обучение на основе корректировок: Используя VBA, таблица сопоставления автоматически обновляется на основе пользовательских отзывов и корректировок, улучшая качество поиска для будущих запросов.
Преимущества
Упрощение рутинных задач: Сокращает усилия на ручное сопоставление данных.
Повышенная точность: Алгоритмы нечеткого поиска улучшают точность результатов.
Автоматическое обучение: Инструмент запоминает предыдущие корректировки и становится умнее с каждым использованием.
Примеры использования
Сопоставление поисковых запросов в прайс-листах.
Обработка специфических наименований товаров для расчета скидок.
Упрощение работы с большими объемами данных с различными вариациями имен.
Требования
Microsoft Excel 2019 или более поздняя версия.
Активированная функция Power Query.
Включенные макросы для запуска скриптов VBA.
Как использовать
Убедитесь, что инструмент находится в отдельной директории:
Перед использованием инструмента рекомендуется убедиться, что инструмент находится в отдельной директории. Важно, чтобы необходимые папки были присутствовали в соответствии с описанной структурой.

Подготовьте Excel файл в папке search:
Поместите ваш Excel файл в папку search, где каждое значение для поиска указано в первом столбце на любом количестве листов.

Обновите данные в Excel файле:
После помещения файла Excel обновите данные в файле (щелкните правой кнопкой мыши на таблице поиска -> обновить). Инструмент автоматически сопоставит данные с последним измененным файлом в папке поиска. Для строк, где соответствия не найдены, дважды щелкните на ячейке [code], чтобы вызвать форму поиска для нахождения нужной позиции. Для версии searching_eg все ранее настроенные вычисления также будут выполняться автоматически.

Просмотрите и скорректируйте сопоставления в таблице поиска:
В таблице поиска просмотрите результаты работы инструмента и убедитесь, что пары сопоставлены правильно. При необходимости вручную внесите корректировки. Затем нажмите кнопку UpdateFromTo или закройте файл. По умолчанию, настройка updateFromToOnClose на листе "settings" установлена в значение 1, что запускает функцию обновления таблицы каждый раз при закрытии файла.

Наслаждайтесь упрощенным сопоставлением:
Наслаждайтесь преимуществами автоматического сопоставления данных, которые были введены хотя бы один раз.
